# Codedoc frontend

Next.js 16 (App Router, TypeScript, Tailwind 4) client for the Codedoc API:
repository ingestion with live pipeline progress, streaming chat with an agent
activity timeline, clickable `file:line` citations with syntax-highlighted code
spans, and a per-answer cost/grounding footer.

Commands: `npm run dev` (needs `NEXT_PUBLIC_API_BASE_URL` in `.env.local`),
`npm run test`, `npm run lint`, `npm run typecheck`. See the root README for the
full-stack setup.
